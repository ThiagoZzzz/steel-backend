# STEEL  - REACT

Este repositorio contiene el proyecto final/entregable realizado para el curso de Desarrollo Web NUCBA, respectivo al módulo de Backend. En esta iteración se implementó una base de datos y su API para acceder a la información de productos, usuarios y órdenes. 

* Autenticación.
* Autorización.
* CRUD de productos.
* CRUD de usuarios.
* CRUD de órdenes.

## ✨ Características Principales

* **Arquitectura backend:** Arquitectura por capas, siguiendo el principio de responsabilidad única.
    * 1. Capa de acceso (Routes).
    * 2. Capa de comunicación (Controllers).
    * 3. Capa de lógica de negocio (Services).
    * 4. Capa de datos (Models).
* **Arquitectura API:** Se implementó una arquitectura REST para el manejo de los datos.
